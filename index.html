CSS盒子模型

在讲盒子模型前，我们先来了解一下三元素

标签分类

  • 块元素

feature（特征）：独占一行，可以通过css改变宽高
例如：div p ul li ol form address。。。

  • 行元素

feature：内容决定元素所占位置，不可以通过css改变宽高
例如：span strong em a del。。。

  • 行块元素（特殊）

feature：内容决定大小，可以改宽高
img

盒子模型

以块元素标签为例：

html：

<div class="demo">
            LIEFoxLIEFoxLIEFoxLIEFox<br>
            LIEFoxLIEFoxLIEFox
</div>

css样式：

.demo{
    width: 200px;
    height: 100px;
    padding: 10px;
    border: 2px solid red;
    margin: 20px;
}

效果：

这就是一个盒子模型，一般由center（height，width），padding，border，margin组成。右下角的盒子模型，是控制台的调试，可以让我们视图化的检查盒子的大小。

注意：margin不属于盒子里的，所以计算时不用。

复合属性值

  • 一个值代表4个方向
padding: 10px;
  • 4个值 上 右 下 左
padding: 10px 10px 20px 10px;
  • 3个值 上 左右 下
padding: 10px 20px 10px;
  • 2个值 上下 左右
padding: 10px 40px;

定位

position: absolute;

absolute：绝对定位，脱离原来位置进行定位，最近的有定位的父级进行定位，如果没有，那么相对于文档进行定位。

position: relative;

relative：相对定位，保留原来位置进行定位，相对自己原来的位置进行定位。

定位口诀：用relative作为参照物，用absolute进行定位

在平常浏览网页时，有些广告不管你怎么滚动滚动条，都是在规定的位置，是利用广告定位来做到的。

position: fixed;

广告定位，可视区窗口定位，无论屏幕如何滚动都固定不动

我们自己来实现一下这个功能

.demo1{
    position: fixed;
    left: 50%;
    top:50%;
    width: 100px;
    height: 100px;
    background-color: #f40;
    margin-top: -50px;
    margin-left: -50px;
}

注意：

  • left、top设置成50%的作用是为了，当用户改变的浏览器的大小时，永远在固定位置（最中间）
  • margin-top、margin-left设置成-50，是因为计算机定位是在方块的左上角，所以要减掉相应的

CSS基础属性

字体修饰

font-size: 30px;
font-weight: normal;
font-style: normal;
font-family: arial;
color: #f40;

font-size设置字号，设置字体大小 默认16 一般用12/14，设置字体的时候实际上就是设置字体的高；

font-weight设置字体粗细，可设置的加粗属性：

bold 加粗
lighter 最细
normal 正常
bolder 特别粗
100-900（也是设置粗细） 数值型设置

font-style可以设置字体斜体

italic斜体 等同em

font-family设置字体，补充：最常用字体 arial -乔布斯发明

color设置字体颜色，有三种设置方法：

1.土鳖式 纯英文单词 red green blue。。
2.颜色代码：#ff00ff
3.颜色函数：rgb（255，255,0）

边框修饰

       边框宽度 边框类型 边框颜色
border:  10px    solid   red;可以拆解开

10px是设置边框的粗细

solid是设置边框的类型，有三种：

solid 线性
dotted 虚线-点型
dashed 虚线-条型

red边框颜色

border复合属性

    width: 10px;
    height: 10px;
    border-left-color: red;
    border-top-color: blue;
    border-right-color: green;
    border-bottom-color: transparent;

补充：transparent：透明色

文本修饰

.demo{
    line-height: 20px;
    text-indent: 2em;
    text-decoration:underline;
    cursor: copy;
}

line-height是行高（行间距）

text-indent是文本缩进，解决首行缩进2个字符，而属性值的单位 em是 1font-size，也就是说，字号是16px，那么，1 em=16 px，这样的话，以后需要改字号大小，就不需要修改缩进的值了

text-decoration 是文本修饰，常设置的属性是 underline（下划线）

cursor是鼠标修饰，可以设置光标移动到文字上，会出现特定的效果

伪类选择器

格式：

选择器 ：hover{

}
.demo:hover{
    height: 200px;
    width: 400px;
    line-height: 100px;
    text-indent: 2em;
    background-color: #f40;
    border-radius: 50px;  设置圆角
    text-decoration:underline;

}

当鼠标移动到修饰的文字上，就会有伪类效果

CSS初级

HTML在平常开发的时候用作于网页的框架搭建，相当于大楼的框架，而CSS是大楼的装修。

引入CSS

1.行间样式

<div style="width: 1000px; background-color: aqua;height: 20px;">234</div>

在标签中写入 style 并设置对应css样式。

2.页面级CSS

在 <herd>里写一对 <style>标签，type="text/css"告诉浏览器是css样式，但可写可不写。

<style type="text/css">
            div{
                background-color: burlywood;
            }
</style>

3.外部CSS文件

在<herd>里写上 <link>标签，在 href里填入css文件路径

<link rel="stylesheet" href="lessn1.css">

CSS选择器

以在css外链式操作

1.id选择器

其特点是一对一。

.html文件

<div id="only">234</div>

.css文件

#开头加上id

#only{
    background-color: aqua;
}

2.类选择器

类选择器特点是多对多

class选择器可以拥有一个或多个类选择器，类选择器中空格隔开，

.html文件

<div class="demo demo1">234</div>

.css文件

.开头

.demo{
    background-color: aqua
}
.demo1{
    width: 100px;
}

3.标签选择器

可以直接用标签选择,HTML文件中全部div表签都可生效

.html文件

<div>234</div>

.css文件

div{
    background-color: blue;
}

4.通配符选择器

以 *来选择，可以选择全部

补充：通配符的应用是初始化所有标签最好的一种方式

*{
    background-color: blue;
}

5.选择器的优先级

important > 行间样式 > id > class|属性 > 标签选择器 > 通配符

6.CSS权重

！important Infinity（正无穷大）
行间样式 1000
id 100
class|属性|伪类 10
标签|伪元素 1
通配符 0

注意：css权重是以256进制进行计算的

CSS复杂选择器

父类选择器

选择标签可以加很多修饰条件，但是只能修饰父子条件，父子选择器的每一个层级都不需要必须是标签，
可以用类标签

举个栗子：

.html

<div class="demo">
            <span>234</span>
</div>

.css

div span{
    background-color: blue;
}

也可

.demo span{
    background-color: blue;
}

子元素选择器

div > span {
    background-color: red;
}

并列选择器

用多个限制条件选择一个选择器并且不加空格

div.demo{
    background-color: red;
}

分组选择器

.html

<div class="demo">
    <strong>
            <span>234</span>
    </strong>
</div>

.css

div,
strong,
span{
    background-color: red;
}

package Dao;

import POJO.User;

import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

/**
 * Created by zjh on 2020/12/24 下午 4:29
 */
public class UserDaoImpl implements UserDao {
    //定义一个Map集合，作为一个数据库存取数据
    private static Map<Integer, User> users;

    static {
        users = new HashMap<Integer, User>();
        users.put(1, new User(1, "小明", 20));
        users.put(2, new User(2, "小王", 30));
        users.put(3, new User(3, "小刚", 18));
        users.put(4, new User(4, "小牛", 27));
    }

    public static Integer initId = 10;

    @Override
    public void add(User user) {
        if (user.getId() == 0) {
            user.setId(initId);
        }
        if (user.getUsername() == null) {
                user.setUsername("请输入姓名");
        }
        users.put(initId, new User(user.getId(), user.getUsername(), user.getAge()));
        initId++;
    }

    @Override
    public void deleteById(Integer id) {
        users.remove(id);
    }

    @Override
    public void updateById(User user) {
        users.put(user.getId(), new User(user.getId(), user.getUsername(), user.getAge()));
    }

    @Override
    public List<User> query() {
        List<User> user = new ArrayList<User>(users.values());
        return user;
    }

    @Override
    public User queryById(Integer id) {
        User user = users.get(id);
        return user;
    }
}

public class User {
    private Integer id;
    private String username;
    private Integer age;

    public User() {
    }

    public User(Integer id, String username, Integer age) {
        this.id = id;
        this.username = username;
        this.age = age;
    }
    public Integer getId() {
        return id;
    }

    public void setId(Integer id) {
        if (id == 0 ){
            this.id = UserDaoImpl.initId;
        }else {
            this.id = id;
        }
    }

    public String getUsername() {
        return username;
    }

    public void setUsername(String username) {
        if (username == ""){
            this.username = "请输入姓名";
        }else {
            this.username = username;
        }
    }

    public Integer getAge() {
        return age;
    }

    public void setAge(Integer age) {
        if (age == 0){
            this.age = 18;
        }else {
            this.age = age;
        }

    }

    @Override
    public String toString() {
        return "User{" +
                "id=" + id +
                ", username='" + username + '\'' +
                ", age=" + age +
                '}';
    }
}

package Util;

import org.apache.commons.beanutils.BeanUtils;

import java.util.Map;

/**
 * Created by zjh on 2020/12/24 下午 4:30
 */

public class WebUtils {
    /**
     * 把 Map 中的值注入到对应的javabean属性中
     * @param value
     * @param bean
     */
    public static <T> T copyParamentBean(Map value, T bean) {
        try {
            System.out.println("注入之前 ：" + bean);
            /**
             * 把所有的参数都注入 user 对象中
             */
            BeanUtils.populate(bean, value);
            System.out.println("注入之后" + bean);
        } catch (Exception e) {
            e.printStackTrace();
        } finally {
        }
        return bean;
    }

    /**
     * 将字符串转换为 int类型
     *
     * @param string
     * @param defaultValue
     * @return
     */
    public static int parseInt(String string, int defaultValue) {
        try {
            return Integer.parseInt(string);
        } catch (Exception e) {
//            e.printStackTrace();
        }
        return defaultValue;
    }
}

package Web;

import javax.servlet.Servlet;
import javax.servlet.ServletException;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import java.io.IOException;
import java.lang.reflect.Method;
import java.util.Base64;

/**
 * Created by zjh on 2020/12/24 下午 5:08
 */
@WebServlet(name = "BaseServlet", urlPatterns = "/baseServlet")
public class BaseServlet extends HttpServlet {
    protected void doPost(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
        //解决 post 请求中乱码的问题
        response.setCharacterEncoding("UTF-8");
        //解决浏览器乱码问题
        response.setContentType("text/html;charset=UTF-8");
        String action = request.getParameter("action");
        try {
            // 获取action业务鉴别字符串，获取相应的业务 方法反射对象
            Method method = this.getClass().getDeclaredMethod(action, HttpServletRequest.class, HttpServletResponse.class);
            System.out.println(method);
            // 调用目标业务 方法
            method.invoke(this, request, response);
        } catch (Exception e) {
            e.printStackTrace();
        } finally {
        }
    }

    protected void doGet(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
        doPost(request, response);
    }
}

package Web;

import POJO.User;
import Service.UserService;
import Service.UserServiceImpl;
import Util.WebUtils;

import javax.servlet.ServletException;
import javax.servlet.annotation.WebServlet;
import javax.servlet.http.HttpServlet;
import javax.servlet.http.HttpServletRequest;
import javax.servlet.http.HttpServletResponse;
import java.io.IOException;
import java.util.List;

/**
 * Created by zjh on 2020/12/24 下午 5:08
 */
@WebServlet(name = "UserServlet", urlPatterns = "/userServlet")
public class UserServlet extends BaseServlet {
    private UserService userService = new UserServiceImpl();

    protected void addUser(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        User user = WebUtils.copyParamentBean(req.getParameterMap(), new User());
        userService.addUser(user);
        resp.sendRedirect(req.getContextPath()+"/userServlet?action=page");

    }

    protected void deleteUserById(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        int id = WebUtils.parseInt(req.getParameter("id"), 0);
        userService.deleteUserById(id);
        req.getRequestDispatcher("/userServlet?action=page").forward(req,resp);
    }

    protected void updateUserById(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        User user = WebUtils.copyParamentBean(req.getParameterMap(), new User());
        userService.updateUserById(user);
        req.getRequestDispatcher("/userServlet?action=page").forward(req,resp);
    }

    protected void queryUserById(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        int id = WebUtils.parseInt(req.getParameter("id"), 0);
        User user = userService.queryUserById(id);
        req.setAttribute("user" , user );
        req.getRequestDispatcher("/pages/user/userAdd.jsp").forward(req,resp);

    }

    protected void page(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        List<User> users = userService.queryUser();
        req.setAttribute("user" , users );
        req.getRequestDispatcher("/pages/user/userList.jsp").forward(req,resp);
    }
}
