CSS初级

HTML在平常开发的时候用作于网页的框架搭建，相当于大楼的框架，而CSS是大楼的装修。

引入CSS

1.行间样式

<div style="width: 1000px; background-color: aqua;height: 20px;">234</div>

在标签中写入 style 并设置对应css样式。

2.页面级CSS

在 <herd>里写一对 <style>标签，type="text/css"告诉浏览器是css样式，但可写可不写。

<style type="text/css">
            div{
                background-color: burlywood;
            }
</style>

3.外部CSS文件

在<herd>里写上 <link>标签，在 href里填入css文件路径

<link rel="stylesheet" href="lessn1.css">

CSS选择器

以在css外链式操作

1.id选择器

其特点是一对一。

.html文件

<div id="only">234</div>

.css文件

#开头加上id

#only{
    background-color: aqua;
}

2.类选择器

类选择器特点是多对多

class选择器可以拥有一个或多个类选择器，类选择器中空格隔开，

.html文件

<div class="demo demo1">234</div>

.css文件

.开头

.demo{
    background-color: aqua
}
.demo1{
    width: 100px;
}

3.标签选择器

可以直接用标签选择,HTML文件中全部div表签都可生效

.html文件

<div>234</div>

.css文件

div{
    background-color: blue;
}

4.通配符选择器

以 *来选择，可以选择全部

补充：通配符的应用是初始化所有标签最好的一种方式

*{
    background-color: blue;
}

5.选择器的优先级

important > 行间样式 > id > class|属性 > 标签选择器 > 通配符

6.CSS权重

！important Infinity（正无穷大）
行间样式 1000
id 100
class|属性|伪类 10
标签|伪元素 1
通配符 0

注意：css权重是以256进制进行计算的

CSS复杂选择器

父类选择器

选择标签可以加很多修饰条件，但是只能修饰父子条件，父子选择器的每一个层级都不需要必须是标签，
可以用类标签

举个栗子：

.html

<div class="demo">
            <span>234</span>
</div>

.css

div span{
    background-color: blue;
}

也可

.demo span{
    background-color: blue;
}

子元素选择器

div > span {
    background-color: red;
}

并列选择器

用多个限制条件选择一个选择器并且不加空格

div.demo{
    background-color: red;
}

分组选择器

.html

<div class="demo">
    <strong>
            <span>234</span>
    </strong>
</div>

.css

div,
strong,
span{
    background-color: red;
}
