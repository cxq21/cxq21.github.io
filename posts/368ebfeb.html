（1）选择结构语句

if语句

if语句格式是

if(条件语句){
    代码块
}

条件语句里是只可以放布尔值，当为true是，执行代码块里的，反之。举个栗子：

 int i=1;
        if(i>0){
            i++;
        }
        System.out.println(i);//2

if…else语句

当判断条件为true时，执行if语句的。如果为false执行else里的。

语法格式

if(判断条件){
    执行语句
}else{
    执行语句
}

举个栗子

 int i=1;
        if(i>2){//false
            System.out.println(i);
        }else {
            System.out.println("执行else");//打印执行else
        }

还有一种三元运算，类似if…else：

判断条件？表达式1：表达式2

当判断条件为true时，执行冒号前面的，反之，执行冒号后面的。

if…else if…else语句

该语句根据多种判断结果，执行相应结果。

语法格式

if(判断条件){
    执行语句
}else if(判断条件){
    执行语句
}else{
    执行语句。。。
}

举个栗子

int i = 10;
if(i>11){
    sout("执行语句");
}else if(true){
    sout("LIEFox");//最后打印出来的结果是LIEFox
}else{
    sout("FOX");
}

switch语句

语法格式

switch (表达式){
           case 1:
               执行语句1;
               break;
           case 2:
               执行语句2;
               break;
           case 3:
               执行语句3;
               break;
           default:
               执行语句4;
               break;
       }

如果不break，将不会结束语句，会执行下一个语句。举个栗子

switch (2){
           case 1:
               sout(1);
               break;
           case 2:
               sout(2);
           case 3:
               sout(3);
               break;
           default:
               sout(4);
               break;
       }
/*结果为23。因为表达式为2，就从case 2 开始执行，
打印2、又因为没有break，不会跳出循环，就会继续执
行case 3 里的语句，直到break跳出语句。*/

switch在JDK7.0的时候，加入了可以判断String类型。

switch中还加入了简单的写法，因为有些情况，case1、2、3都有可能结果相同，所以可以这样：

switch (2){
           case 1:
           case 2:
           case 3:
               sout(3);
               break;
           default:
               sout(4);
               break;
       }//结果为3

（2）循环结构语句

while语句

语法格式

while (循环条件){
    执行语句
}

循环条件里依旧，true就执行，false就停止。

举个栗子

int i =1;
       while (i<=5){
           System.out.println("i="+i);
           i++;
       }
       结果/*i=1
            i=2
            i=3
            i=4
            i=5*/

do…while循环语句

语法格式

do{
    执行语句
}while(循环条件)

可见，do while循环会至少循环一次。

我们把对1到100的求和用do while循环改写一下：

int sum = 0;
        int n = 1;
        do {
            sum = sum + n;
            n ++;
        } while (n <= 100);
        System.out.println(sum);//结果为5050

小结

do while循环先执行循环，再判断条件；

do while循环会至少执行一次。

for语句

语法格式

for (初始化表达式;循环条件;操作表达式){
    执行语句
}

举个栗子

for (int i =0;i<=7;i++){
      System.out.println("LIEFox");
}//将会打印7次LIEFox

执行顺序是这样的》

1.起始位置从i=0开始，“0<=7”为true，打印LIEFox，回到i++,i=1。
2.i=1,"1<=7"为true，打印LIEFox，回到i++,i=2。
。。。。。
7.i=7,"7<=7"为true，打印LIEFox，回到i++,i=8。
8.i=8,"8<=7"为false，循环结束。

循环嵌套

所谓循环嵌套，指的是多种控制语句相组合，实现自己想要的功能。

就简单的两层for循环，就可以实现一个美妙的形状组成的实现。

        for (int i =0;i<=10;i++){
            for (int x =i;x<=5;x++){
                 System.out.print("*");
            }
            System.out.println();
        }

跳转语句

break和continue

break：在之前的switch控制语句就用到了，执行到break，就结束循环。

continue：是跳出本次循环。

举个continue的栗子

public class Main {
    public static void main(String[] args) {
        int sum = 0;
        for (int i=1; i<=5; i++) {
            System.out.println("begin i = " + i);
            if (i % 2 == 0) {
                continue; // continue语句会结束本次循环
            }
            sum = sum + i;
            System.out.println("end i = " + i);
        }
        System.out.println(sum); // 9
    }
}
//结果
begin i = 1
end i = 1
begin i = 2
begin i = 3
end i = 3
begin i = 4
begin i = 5
end i = 5
9

注意观察continue语句的效果。当i为奇数时，完整地执行了整个循环，因此，会打印begin i=1和end i=1。在i为偶数时，continue语句会提前结束本次循环，因此，会打印begin i=2但不会打印end i = 2。

在多层嵌套的循环中，continue语句同样是结束本次自己所在的循环。

小结

break语句可以跳出当前循环；

break语句通常配合if，在满足条件时提前结束整个循环；

break语句总是跳出最近的一层循环；

continue语句可以提前结束本次循环；

continue语句通常配合if，在满足条件时提前结束本次循环。
