CSS盒子模型

在讲盒子模型前，我们先来了解一下三元素

标签分类

  • 块元素

feature（特征）：独占一行，可以通过css改变宽高
例如：div p ul li ol form address。。。

  • 行元素

feature：内容决定元素所占位置，不可以通过css改变宽高
例如：span strong em a del。。。

  • 行块元素（特殊）

feature：内容决定大小，可以改宽高
img

盒子模型

以块元素标签为例：

html：

<div class="demo">
            LIEFoxLIEFoxLIEFoxLIEFox<br>
            LIEFoxLIEFoxLIEFox
</div>

css样式：

.demo{
    width: 200px;
    height: 100px;
    padding: 10px;
    border: 2px solid red;
    margin: 20px;
}

效果：

这就是一个盒子模型，一般由center（height，width），padding，border，margin组成。右下角的盒子模型，是控制台的调试，可以让我们视图化的检查盒子的大小。

注意：margin不属于盒子里的，所以计算时不用。

复合属性值

  • 一个值代表4个方向
padding: 10px;
  • 4个值 上 右 下 左
padding: 10px 10px 20px 10px;
  • 3个值 上 左右 下
padding: 10px 20px 10px;
  • 2个值 上下 左右
padding: 10px 40px;

定位

position: absolute;

absolute：绝对定位，脱离原来位置进行定位，最近的有定位的父级进行定位，如果没有，那么相对于文档进行定位。

position: relative;

relative：相对定位，保留原来位置进行定位，相对自己原来的位置进行定位。

定位口诀：用relative作为参照物，用absolute进行定位

在平常浏览网页时，有些广告不管你怎么滚动滚动条，都是在规定的位置，是利用广告定位来做到的。

position: fixed;

广告定位，可视区窗口定位，无论屏幕如何滚动都固定不动

我们自己来实现一下这个功能

.demo1{
    position: fixed;
    left: 50%;
    top:50%;
    width: 100px;
    height: 100px;
    background-color: #f40;
    margin-top: -50px;
    margin-left: -50px;
}

注意：

  • left、top设置成50%的作用是为了，当用户改变的浏览器的大小时，永远在固定位置（最中间）
  • margin-top、margin-left设置成-50，是因为计算机定位是在方块的左上角，所以要减掉相应的

奥运五环案例

做一个奥运五环，永远在最中间

HTML：

<!DOCTYPE html>
<html lang="en">

    <head>
        <meta charset="utf-8">
        <title>标题</title>
        <link rel="stylesheet" href="lessn1.css">
    </head>

    <body>
        <div class="five">
            <div class="blue"></div>
            <div class="yellow"></div>    
            <div class="red"></div>    
            <div class="black"></div>    
            <div class="green"></div>
            </div>
                <br>*100
    </body>

</html>

css：

.five{
    position: fixed;
    left: 50%;
    top: 50%;
    width: 500px;
    height: 250px;
    background-color: transparent;

    margin-left: -250px;

    margin-top: -125px;
}
.blue{

position: absolute;

left:65px;

top:30px;

width: 100px;

height:100px;

border: 10px solid blue;

border-radius: 50%;

z-index: 1;

}

.yellow{

position: absolute;

left:190px;

top:30px;

width: 100px;

height:100px;

border: 10px solid #FFFF00;

border-radius: 50%;

z-index: 3;

}

.black{

position: absolute;

right:65px;

top:30px;

width: 100px;

height:100px;

border: 10px solid black;

border-radius: 50%;

z-index: 5;

}

.green{

position: absolute;

left:127px;

top:100px;

width: 100px;

height:100px;

border: 10px solid green;

border-radius: 50%;

z-index: 2;

}

.red{

position: absolute;

right:127px;

top:100px;

width: 100px;

height:100px;

border: 10px solid red;

border-radius: 50%;

    z-index: 4;

}
